## 💻 Projects (주요 개발 경험)

학부 과정 및 대회를 통해 진행한 핵심 프로젝트들입니다.

### **[IoT/Embedded AI] SOOM: 수면 비서 IoT 시스템 (2025) - 🏆 LG CTO상**

* **내용:** 수면 패턴 분석 및 환경 제어를 위한 AI 기반 IoT 시스템 개발.

* **기술 스택:** Python, ESP32, AI Model Deployment, IoT

### **[AI Application] POOM: WiFi Sensing 기반 낙상 감지 (2025) - 🥈 유니톤 우수상**

* **내용:** WiFi CSI 데이터를 활용하여 비접촉 방식으로 낙상 사고를 감지하는 머신러닝 모델 개발.

* **기술 스택:** Python, WiFi CSI Data Processing, ML (Machine Learning)

### **[Deep Learning/CV] 멀티모달 자율주행 AI (2025-2)**

* **내용:** 자율주행을 위한 CLIP distill 및 Talk2Car 데이터셋을 활용한 경량화 모델 파인튜닝.

* **기술 스택:** PyTorch, CLIP, Knowledge Distillation

### **[Deep Learning/CV] 데이터사이언스 프로젝트 (2025-2)**

* **내용:** DDGAN을 기반으로 KITTI 데이터셋에 대한 Amodal Segmentation Task 수행.

* **기술 스택:** PyTorch, GAN (Generative Adversarial Network)

### **[Web/CV] SSCCounter: 동방 인원 수 YOLO 기반 웹 앱 (2025)**

* **내용:** YOLOv8 모델을 활용하여 동아리방 실시간 인원 수를 카운트하고 WebSocket으로 웹에 표시하는 애플리케이션.

* **기술 스택:** FastAPI, YOLOv8

### **[Embedded AI] 임베디드 스마트팜 (2025-2)**

* **내용:** 컴퓨터 비전 기반으로 잎의 건강 상태 및 병을 판단하는 AI 모델을 임베디드 환경에 적용.

* **기술 스택:** Python, Pytorch, Tflite
